# Javascript - Variáveis e Constantes

- Variáveis
    > - var
    > - let
    > - const

## Exemplos

- Destructuring Assignment

    ~~~javascript
    var [apple, banana, orange] = ["Apple", "Banana", "Orange"];
    console.log(apple);
    ~~~
    
- Symbols - generate a unique identifier 

    ~~~javascript
    const uniqueId = Symbol('valueToDebug');
    console.log(uniqueId);
    ~~~
